Don Charly's Mexican Catering

This is a catering website for my first client using HTML and CSS. Click the title to view the live website. This project was a joy to develop from start to finish. Each part of the website is modular. For example, the header and footer can be placed on any page and the style and font will not be changed. I enjoyed implementing a dropdown menu on the MENU navigation heading to link to the specific items in the menu page. The most challenging part of the website was getting the contact page cards in the exact position I wanted them along with their contents. To host the website for the client I am using Google's Firebase Hosting.
